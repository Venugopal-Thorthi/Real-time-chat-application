# Real-Time Chat Application

## Features
- Real-time messaging
- Basic username login
- Chat history on join
- Responsive frontend design
- Easy scalability with socket.io events

## Tech Stack
- Node.js + Express.js (Backend)
- React.js (Frontend)
- Socket.IO (WebSocket communication)
